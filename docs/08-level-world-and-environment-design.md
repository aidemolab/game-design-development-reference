# 08. Level, World, and Environment Design

## Standard

The environment is part of the rules. It guides movement, reveals objectives, controls visibility, supports pacing, and creates strategic choices.

## Evidence

- Boat Attack demonstrates a coherent URP environment, water presentation, camera framing, and optimisation.
- PUBG demonstrates how terrain, cover, routes, loot locations, and a changing safe area shape encounters.
- Stronghold Crusader and Age of Empires show how resources and defensible geography influence strategy.
- Anodyne 2 provides documentation connecting scenes, levels, and project folders in a released Unity game.

## Best practices

- Block out levels with simple geometry before producing final art.
- Use landmarks, lighting, colour, sound, and composition for navigation.
- Offer meaningful routes rather than decorative space.
- Control sight lines, encounter distance, and safe areas deliberately.
- Test traversal time and dead space.
- Separate visual boundaries from collision boundaries and verify both.

## Originality boundary

Learn how maps shape decisions, but create original layouts, locations, routes, landmarks, themes, and encounter structures.
