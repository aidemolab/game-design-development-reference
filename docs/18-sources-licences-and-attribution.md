# 18. Sources, Licences, and Attribution

## Evidence policy

This knowledge base contains original summaries and transferable principles. It links to source material rather than reproducing substantial code, documentation, artwork, audio, maps, or proprietary content.

Public visibility is not a licence. Always inspect the current repository licence and third-party notices before copying, modifying, or redistributing material.

## Attribution standard

Every derived study note should name and link its source, state whether the lesson concerns design or implementation, record the applicable licence when code is reused, and describe the original application separately.
